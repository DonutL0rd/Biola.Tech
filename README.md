# biola.tech

A student-run initiative exploring technology, innovation, and community at Biola University.

## Overview

This is a simple, modern landing page for the biola.tech domain. It serves as a placeholder while the project is in development, clearly communicating that this is a work-in-progress student project.

## Features

- **Clean, modern UI** with Biola-inspired red and black color scheme
- **Responsive design** that works on all device sizes
- **Clear messaging** about student-run status and work-in-progress nature
- **Direct contact link** via email badge
- **Minimal dependencies** - pure HTML/CSS, no frameworks required
